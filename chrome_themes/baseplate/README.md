# Baseplate Chrome Theme

A dark Chrome theme for baseplate.sh

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" in the top right
3. Click "Load unpacked"
4. Select this directory

## Customization

Edit `manifest.json` to customize colors. All colors use RGB format: `[R, G, B]` where values are 0-255.

For a complete reference of all available color properties and what they control, see `COLOR_REFERENCE.md`.

## Background Image

Replace `images/background.png` with your own background image. Recommended sizes:
- Minimum: 1920x1080
- Recommended: 2560x1440 or higher
